# Virtual Display over USB

Turn your Android tablet into a second display for MacBook via USB-C.

## One-Command Setup

```bash
./setup.sh
```

## Requirements

- macOS 14+ (Sonoma)
- Android 8+ (API 26)
- USB-C cable

## Features

**macOS:**
- Settings window with Apple design
- Configure resolution, FPS, bitrate
- Real-time performance stats
- Menu bar integration

**Android:**
- Modern Material Design 3 UI
- Touch control support
- Hardware H.265 decoding
- Real-time FPS overlay

## Uninstall

```bash
./scripts/uninstall.sh
rm -rf VirtualDisplay.app
```
